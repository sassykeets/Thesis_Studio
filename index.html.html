1. What is Thesis Studio?

Thesis Studio analyses your thesis Word document and shows structure, word counts, writing health, and helps you plan — all locally in your browser. Your file never leaves your computer.

✓ No installation✓ File stays local✓ .docx only✓ Free
Thesis Studio reads heading styles from your Word document. You must use Word's built-in Heading 1, Heading 2, and Heading 3 styles — not just bold text or large fonts.
2. How to Format Your Word Document

To apply a heading style: select text → click the style name in the Styles panel on the Home tab. Do not just make text bold.

StyleUsed forExample
Heading 1Chapters, References, AppendixCHAPTER 1 INTRODUCTION
Heading 2Sections within a chapter1.1 Background and Motivation
Heading 3Subsections1.1.1 The Role of AI
Chapter heading rule
✓ CHAPTER 1 INTRODUCTION
✓ Chapter 2: Scoping Review
✗ Introduction   ← missing "Chapter"
✗ 1. Introduction ← missing "Chapter"
References and Appendix rule

Any Heading 1 containing References, Bibliography, Appendix, or Closing triggers exclusion — everything after is removed from your word count and shown in the Appendix Viewer.

[Heading 1] CHAPTER 1 INTRODUCTION      ← chapter ✓
[Heading 2]   1.1 The Story of Sunita
[Heading 3]     1.1.1 Background
[Heading 2]   1.2 Research Questions

[Heading 1] CHAPTER 2: SCOPING REVIEW     ← chapter ✓
[Heading 2]   2.1 Introduction

           ... more chapters ...

[Heading 1] References                    ← excluded ✓
[Heading 1] Appendix                     ← excluded ✓
[Heading 2]   Appendix A: Survey Instrument
3. Features
📄 Structure
Full collapsible tree — word counts, reading time, % per section. Set targets per chapter.
↻ Repetitions
Repeated 3/4/5-word phrases across body text.
⚠ Weasel Words
Overused academic adverbs in 5 categories.
📎 Citations
Counts (Author, Year) and [n] patterns.
🩺 Sentence Health
Sentence length, passive voice %, repeated openers.
✏️ Spelling
Common misspellings, confusable pairs, double spaces.
🔤 Abbreviations
Detects uppercase abbreviations, flags undefined ones.
🧮 Word Count
Full DPhil-standard calculation with excluded components.
📖 Reader
Serif reader with text-to-speech, voice and speed controls.
📎 Appendix Viewer
Full structure of excluded content — sections, tables, images.
📅 Chapter Planner
4-step wizard: audit sections, calendar, PDF download.
4. What Thesis Studio Does Not Do
✗
Full grammar checking — Use Grammarly or Word's grammar checker
✗
Citation verification — Counts patterns only — does not check if citations are real
✗
Plagiarism detection — Use Turnitin or your institution's tool
✗
AI-powered analysis — All rule-based — no AI calls made
✗
PDF or Google Docs files — .docx only — convert first
✗
Old .doc format — Open in Word and Save As .docx
5. Tips for Best Results
✦
Use Word's built-in Heading styles — bold text is not the same
✦
Include the word Chapter in all chapter headings
✦
Make References and Appendix Heading 1 style
✦
Set chapter targets — defaults are based on 50,000 words
✦
Upload regularly as you write — analysis reruns each time
✦
Best audio quality: use Microsoft Edge on Windows
✦
If a chapter shows 0 words — check it uses Heading 1 and contains "Chapter"
